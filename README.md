# Insight Dashboard

Panel de monitorización y análisis de estado del sistema.

## Descripción
`insight_dashboard` es una herramienta centralizada para visualizar métricas críticas y el estado operativo de los servicios del sistema OpenClaw y su entorno.

## Características
- Monitorización de servicios activos.
- Visualización de logs de errores del sistema.
- Seguimiento de estados de gateways y conexiones.

## Desarrollo
Este proyecto sigue la metodología **git-flow**. Los cambios deben proponerse mediante Pull Requests a la rama `develop`.
